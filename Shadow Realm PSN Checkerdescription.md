# 🌑⚔️ Shadow Realm PSN Checker ⚔️🌑
**PlayStation Network Account Hit Detector**
⚡ Fast • Optimized • Stable • Premium

---

## 🚀 Overview
Shadow Realm PSN Checker is a multi-threaded checker that logs into Hotmail/Outlook accounts from a combo list and scans inboxes for PlayStation Network emails. It captures PSN account details and order information for all valid hits and saves them to organized result files.

---

## 🎮 Features

### 🎮 Capture Features
- PlayStation Network email detection
- PSN order capture
- Sony account detail extraction
- Multiple Sony email patterns supported
- Full inbox access validation

### ⚙️ System Features
- ⚡ ThreadPoolExecutor multi-threaded engine
- 📊 Live hit counter
- 📁 Organized result files per capture type
- 🔄 Smart retry on API errors
- 🌑 Clean terminal output
